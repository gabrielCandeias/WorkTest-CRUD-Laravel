O Web site conta com o registro, exclusão, busca, e edição (CRUD), além das telas administrativas, login, etc. Utilizando o bootstrap para dar mais estilo ao site, junto com Javascript JQuery Ajax, para realizar funções dinâmicas como o select de cidade e estados. Além de utilizar o pacote do Breeze que traz consigo diversos mecanismos de autenticação e autorização, trazendo uma segurança elevada para o site. É utilizado umas das últimas versões do PHP -> 8.1.2, além de ser todo construído pela arquitetura MVC utilizando o laravel. O site ainda possui controle de versão (GIT), onde é possível visualizar as etapas, elevando o controle sobre o desenvolvimento do site.
